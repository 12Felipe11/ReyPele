Hardware y Conexiones

HC-SR04 — Sensor Ultrasonico

TRIG Pin D9
ECHO Pin D8
VCC 5V
GND GND

Rango: 2cm – 400cm | Umbral por defecto: 30cm

SSD1306 — Pantalla OLED 128x64

SDA Pin A4 (I2C)
SCL Pin A5 (I2C)
VCC 3.3V
GND GND

Direccion I2C: 0x3C | Libreria: Adafruit_SSD1306

Protocolo Serial (9600 baud)

DireccionMensajeDescripcion
Arduino → Java DATA:count,dist,pres,luz Datos periodicos cada 500ms
Arduino → Java ENTRY:totalCount Evento inmediato al detectar entrada
Arduino → Java READY Arduino listo tras reinicio
Java → Arduino READ Solicitar lectura inmediata
Java → Arduino ALARM_ON / ALARM_OFF Activar / desactivar alarma
Java → Arduino LIGHT_SET:0-100 Intensidad de luces (porcentaje)
Java → Arduino SET_THRESHOLD:cm Umbral de distancia para deteccion
Ejemplo de DATA:
DATA:12,24.5,1,75 → 12 entradas, 24.5cm, presencia SI, luces 75%

Logica de Conteo

El conteo de entradas usa deteccion por flanco con el sensor ultrasonico:

  1. Persona se acerca → distancia cae por debajo de umbralDistancia (30cm) → presenciaDetectada = true
  2. Persona se aleja → distancia sube por encima del umbral → se registra +1 entrada
  3. Bloqueo de 800ms para no duplicar conteos (HU-03)
